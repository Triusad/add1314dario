Informe

Comenzamos instalando el Servidor de Windows 2008 y seleccionando los componentes necesarios a instalar para establecer el servidor de impresion(administracion del servidor-> roles).

Luego instalamos el "PDF Creator" para utilizarlo de impresora virtual.

En la propia instalacion del PDF Creator nos crea una nueva impresora.

De manera que la marcamos como impresora predeterminada y la compartimos.

Luego creamos un documento para imprimir y lo "imprimimos".

Entrando en la web del servidor y pausando la cola de impresion podemos capturar el documento "enviado a la impresora".
